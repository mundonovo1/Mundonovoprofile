☁ Connect Cloud Sync (one-time setup)

Takes ~2 minutes. Your data will then sync across all devices via Google Sheets — free, no account needed beyond Google.

1

Open script.google.com, click New project, delete any existing code, then paste the script below:

var SHEET_NAME = 'MundoNovo'; function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME) || SpreadsheetApp.getActiveSpreadsheet().insertSheet(SHEET_NAME); var data = sheet.getRange(1,1).getValue(); return ContentService.createTextOutput(data || '{}') .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME) || SpreadsheetApp.getActiveSpreadsheet().insertSheet(SHEET_NAME); sheet.getRange(1,1).setValue(e.postData.contents); return ContentService.createTextOutput('ok') .setMimeType(ContentService.MimeType.TEXT); }
2

Click Deploy → New deployment. Set type to Web app, set Execute as to Me, set Who has access to Anyone. Click Deploy and authorize when prompted.

3

Also open Google Sheets, create a new blank spreadsheet, then go back to Apps Script, click Project Settings (gear icon) and link it to that spreadsheet under Google Sheets.

Tip: name the spreadsheet something like "Mundo Novo Sync" so you can find it later.
